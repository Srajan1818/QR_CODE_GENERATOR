# 🔗 QR Code Generator

A simple and responsive **QR Code Generator** built with **HTML**, **CSS**, and **JavaScript** that instantly generates QR codes for any text or URL you enter.

---


## ✨ Features

- 🔗 Generate QR codes for any text or URL
- ⚡ Instant QR code generation
- 📥 Download the generated QR code
- 🎨 Clean and user-friendly interface

---

## 🎯 Project Purpose

This project was created to practice front-end web development concepts such as:

- 🔄 DOM Manipulation
- 🖱️ Event Handling
- 🚀 JavaScript Fundamentals

---

## 🛠️ Technologies Used

- **HTML5** – Structure and markup
- **CSS3** – Styling and responsive layout
- **JavaScript (ES6+)** – App logic and API integration
- **QR Code API** – For generating QR codes

---




## 📌 How To Use

1. Type any **text or URL** in the input field.
2. Click the **Generate** button.
3. Your QR code will appear instantly.
4. Click **Download** to save the QR code image.
